Pair project for the Tampere University course COMP.SE.200 Software Testing.

[![Coverage Status](https://coveralls.io/repos/github/Real-bay/SoftwareTestingProject/badge.svg?branch=actions)](https://coveralls.io/github/Real-bay/SoftwareTestingProject?branch=actions)

## Project Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (version 14 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Real-bay/SoftwareTestingProject.git
   cd SoftwareTestingProject
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running Tests Locally

### Run All Tests

To execute the entire test suite:

```bash
npm test
```

### Run Tests with Coverage

To run tests and generate a coverage report:

```bash
npm run test:coverage
```

The coverage report will be displayed in the terminal.
